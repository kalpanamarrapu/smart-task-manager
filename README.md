# Smart Task Manager (Flask)

A full-stack productivity web application built using Flask that helps users manage daily tasks, track study time, schedule events, and take notes all in one place.

---

## 🚀 Features

- 📚 **Study Timer**
  - Start/stop timer
  - Tracks study sessions

- ✅ **Task Manager**
  - Add tasks with deadlines
  - Mark tasks as completed
  - Delete tasks

- 📅 **Interactive Calendar**
  - Click on dates
  - Add and view events

- 📝 **Notes System**
  - Create and save multiple notes
  - Delete notes anytime

---

## 🛠 Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **Version Control:** Git & GitHub
- **Deployment:** Render

cd smart-task-manager
pip install -r requirements.txt
python app.py
